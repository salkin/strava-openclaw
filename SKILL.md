# Strava OpenClaw Skill

Fetch and explore your Strava activities through OpenClaw. Retrieve recent workouts, athlete stats, and your last activity using the Strava API.

## Requirements

- Python 3.7+
- `stravalib` package
- Strava account with API credentials (free)

## Setup

### 1. Install dependencies

```bash
pip install stravalib
```

### 2. Run the setup wizard

```bash
python3 setup.py
```

The setup wizard will:
1. Guide you through creating a Strava API application
2. Open a browser for OAuth authorization
3. Save credentials to `~/.strava_credentials.json`

### 3. Create a Strava API App

1. Go to <https://www.strava.com/settings/api>
2. Click **Create & Manage Your App**
3. Fill in:
   - **Application Name:** OpenClaw Strava
   - **Category:** Tool or Analytics
   - **Website:** `http://localhost`
   - **Authorization Callback Domain:** `localhost`
4. Copy your **Client ID** and **Client Secret**

## Commands

### Fetch recent activities

```bash
python3 strava_activities.py fetch
```

Returns the 10 most recent activities with name, type, distance (km), moving time, and date.

### Get athlete stats

```bash
python3 strava_activities.py stats
```

Returns recent running and cycling totals (last 4 weeks).

### Get last activity

```bash
python3 strava_activities.py last
```

Returns details of the most recent activity.

### Fetch activities by date range

```bash
python3 strava_activities.py fetch-range AFTER [BEFORE]
```

Fetches activities from `AFTER` up to (and including) `BEFORE`. Both dates use `YYYY-MM-DD` format. If `BEFORE` is omitted, all activities from `AFTER` to the present are returned.

## Notes

- Credentials are personal and should not be shared
- Strava API rate limits: 100 requests per 15 minutes, 1,000 per day
- The access token is automatically refreshed when it expires
